#Useful Git Commands

when setting up new repository
```

   echo "#Useful Git Commands" >> README.md
   cat README.md 
   git init
   git add .
   git -m "first commit"
   git commit -m "first commit"
   git branch -M main
   git remote add origin git@github.com:amittri1025/UsefulGitCommands.git
   git push -u origin main


```
